# Rails React Demo

### Getting started

- Clone
- Run `docker-compose up`
- Run `yarn install`
- Create db `rails db:create`
- Run migration `rails db:migrate`
- Run seed `rails db:seed`
- Install foreman `gem install foreman`
- Start api and client `foreman start`
